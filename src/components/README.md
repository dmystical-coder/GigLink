# Components

This directory contains the React components for the GigLink application.

## Structure

- **bounty/**: Components specific to the Bounty Card and its sub-elements.
  - `BountyCard.tsx`: The main card container.
  - `BountyCardHeader.tsx`: Title and issuer info.
  - `BountyCardBadges.tsx`: Status and difficulty badges.
  - `BountyReward.tsx`: Reward display.
  - `BountyCardFooter.tsx`: Footer with deadline and applicants.

- **ui/**: Reusable UI primitives (buttons, inputs, etc.).

- **FeedHeader.tsx**: The header for the bounty feed with search and filters.
- **BountyGrid.tsx**: The grid layout for displaying bounties.
- **SearchInput.tsx**: A reusable search input component.
- **FilterDropdown.tsx**: A reusable dropdown filter component.
- **ToggleGroup.tsx**: A reusable toggle button group.
